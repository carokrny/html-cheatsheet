HTML Quick Reference Guide

A quick refernce guide for commonly used HTML tags.

Semantic HTML

Tag Name Description
<article> Article Used to hold content that makes sense independently.
<aside> Aside Used to contain additional information that is not required.
<figure> Figure Used to encapsulate media of any type.
<figcaption> Figure Caption Used within <figure> to provide a description of the media.
<footer> Footer Can be used at the bottom of any container to provide a footer.
<header> Header Can be used at the top of any container to provide a heading.
<main> Main Can be used to designate the main portion of a container.
<nav> Navigation Used to contain navigation links.
<section> Section Used to contain content of the same theme.

HTML Table Tags

Tag Name Description
<table> Table A container for the table content.
<thead> Table Header Nested in <table> to define the column headings of the table.
<tbody> Table Body Nested in <table> to define the body of the table.
<tfoot> Table Footer Nested in <table> to use table rows for footer content.
<tr> Table Row Nested in a <thead>, <tbody>, or <tfoot> element to define a table row.
<th> Table Heading Nested in a <tr> element to add a title to a column.
<td> Table Data Nested in a <tr> element to add data to a column.

Other HTML Tags

Tag Name Description
<a> Anchor Used to include hyperlinks in an HTML document using attribute href.
<audio> Audio Used to embed audio using src to link the audio.
<br> Line Break Creates a line break in text. Self-closing tag.
<div> Division Used as a container that divides the HTML document into related content.
<em> Emphasis Used to emphasize text, usually italics by default.
<embed> Embed Used to embed any media using src. Self-closing tag.
<img> Image Used to embed an image using src to link image. Self-closing tag.
<li> List Item Used to create list items inside <ul> and <ol>.
<ol> Ordered List Used to create a list in sequential order. Default is numbered list items.
<span> Span An inline container to group text for styling purposes.
<strong> Strong Used to highlight text, usually bold by default.
<title> Title Contains text that is displayed in the page's tab in the broswer.
<ul> Unordered List Used to create a list without a particular order. Default is bullet points.
<video> Video Used to embed a video using src to link video.
